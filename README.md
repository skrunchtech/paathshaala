# Paatshaala

A Next.js app with number games, a kid-friendly calculator, and math practice.

## Getting Started

1. Install dependencies:

```bash
pnpm install
```

2. Run the development server:

```bash
pnpm dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project layout

- `app/page.tsx` — games hub
- `app/calculator/` — math calculator
- `app/*/` — individual mini-games
